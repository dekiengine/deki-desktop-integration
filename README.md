# Deki Desktop Integration

Desktop platform HAL (Hardware Abstraction Layer) for the Deki Engine: memory management and filesystem support.

Part of the [Deki Engine](https://github.com/ATeam-Dekiru/deki-engine) module ecosystem.

## Installation

In the Deki Editor Package Manager, add this repository URL.

## Dependencies

None.
